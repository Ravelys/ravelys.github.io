Learn how to use “supposed to” correctly in English. This is a very common phrase we use in English every day. We use “supposed to” to talk about obligation, expectation, and plans. That might sound complicated, but I’ll make it clear in this lesson. I’ll show you the three main ways “supposed to” is used in English, and then we’ll practice together to make sure you’ve got it.
