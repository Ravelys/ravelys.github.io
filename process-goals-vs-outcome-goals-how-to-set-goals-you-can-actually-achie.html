If you want to learn how to set goals in life and actually achieve them, then you need the right system to get you there.

So in this video, we’ll talk about the difference between the two and then you will learn why it’s dangerous to focus on just outcome goals. Plus, we will provide the specific strategies you can use to set process goals that can help you achieve what you want from life.
