Installing the database file
1. Download the file above


2.On Windows Place the file in C:\Users\USERNAME\Documents\Sports Interactive\Football Manager 2022\editor data

On a Mac: Macintosh HD\users\USERNAME\Documents\PUBLIC\Sports Interactive\Football Manager 2022\editor data
OR
Macintosh HD\users\USERNAME\Documents\Sports Interactive\Football Manager 2022\editor data


3. Start a new save, on set up make sure the database ‘Projek_promoted’ is active and ticked as seen here: https://i.imgur.com/bNrsY7g.png

If the database .fmf file doesn’t appear you have not placed it in your editor data correctly.
