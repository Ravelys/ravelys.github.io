In this lesson, you can learn how to make and accept suggestions, using the right phrases and vocabulary. You’ll also see how to reject someone else’s suggestion politely.

This lesson can help you:
- Learn useful phrases to make suggestions to someone in English.
- Understand how you can accept suggestions using common expressions.
- See how you can reject suggestions in English in a polite way.
- Get useful vocabulary and phrases to add a reason to a rejected suggestion in English.
- Have the opportunity to see these phrases used in real dialogue examples.
- Realize what different types of intonation sound like, and how you can use them when accepting or rejecting suggestions in English.
