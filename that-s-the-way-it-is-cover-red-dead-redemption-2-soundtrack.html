This song was the first one I released on my channel when I discovered Red Dead Redemption 2. It’s the song that gave me the boost and visibility that I had been working so hard for the previous 5 years. Since there were some more verses added in the OST release, I though I should probably do a version like that too. I hope you like this new version as much as I do!
