Java have huge number of build in classes in the jre.
While working with JDBC, Servlets and other frameworks you can use Third Party classes.
All the io classes are under java.io Package.
All the networking classes are under java.net package, and all the classes required while working with JDBC are under java.sql Package.
When all these classes are classified in particular format and then put in a Folder,this Folder in Java is called Packages.
When we create a user defined Package,it should have a unique name.
