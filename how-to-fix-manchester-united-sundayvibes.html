On this week’s Sunday Vibes, the team discusses Manchester United’s tricky start to the season - and the potentially fatal run of games Ole has coming up (Liverpool, Tottenham, Man City)!

If it’s the former, how should United go about rebuilding the club (having already spent £400m in the last three years)? If it’s the latter - what changes does Ole need to make to turn things round?
